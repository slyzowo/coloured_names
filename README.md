# coloured_names
A datapack for minecraft that allows you to change your name colour using commands

Example command
```
/name_colour Red
```

All the options are :
- White
- Pink
- Red
- Maroon
- Orange
- Yellow
- Lime
- Green
- Light_Blue
- Teal
- Blue
- Dark_Blue
- Purple
- Grey
- Dark_Grey
- Black
